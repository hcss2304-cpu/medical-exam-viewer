# 医師国家試験ビューアー

第115回医師国家試験の過去問を学習できるWebアプリケーションです。

## 機能

- **全399問収録**（A〜F問題）
- **科目別フィルター**（23科目）
- **問題区分フィルター**（A〜F）
- **キーワード検索**
- **正答率表示**
- **画像表示**（計266枚）
- **ランダム出題モード**
- **PWA対応**（スマホでアプリのように使用可能）
- **レスポンシブデザイン**（スマホ対応）

## 使い方

### 方法1: ローカルで開く
`index.html` をブラウザで開くだけで使用できます。

### 方法2: GitHub Pages で公開
1. このリポジトリをFork
2. Settings → Pages → Source を `main` ブランチに設定
3. 数分後に `https://<username>.github.io/<repo-name>/` でアクセス可能

### 方法3: PWAとしてインストール
ブラウザで開いた後、「ホーム画面に追加」でアプリのように使用できます。

## 画面説明

### フィルター
- **科目**: プルダウンから選択（例: 循環器内科、消化器内科、小児科...）
- **問題区分**: A〜F問題を選択
- **キーワード**: 問題文を検索

### 問題表示
- 問題をクリックすると詳細が展開
- 「正答を表示」ボタンで正解と正答率を表示
- 画像がある問題には「画像」バッジが表示

### ランダム出題
「ランダム出題」ボタンで、フィルター条件に合う問題からランダムに出題

## データについて

- **対象**: 第115回医師国家試験（2021年実施）
- **問題数**: 399問（A〜F問題）
- **正答率**: MEC Netから取得
- **画像**: medical-illustration.club から取得

## 注意事項

- 本ツールは学習支援を目的としています
- 問題・正答は厚生労働省公開資料に基づいています
- 画像は医療美術部（medical-illustration.club）から取得しています
